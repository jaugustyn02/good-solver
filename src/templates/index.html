Welcome to GoodSolver!

GoodSolver is a platform for collaborative decision-making using the Analytic Hierarchy Process (AHP). Create decision scenarios, log in as an expert, add surveys using an invite code, and explore rankings of completed scenarios.

Main Features

  • - Create new decision scenarios.
  • - Log in as an expert, add surveys with invite code, and fill them out.
  • - Browse results of completed scenarios, including final rankings.
